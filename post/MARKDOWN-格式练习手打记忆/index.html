MARKDOWN 格式练习手打记忆

（注意所有文段和最后一个格式符之间都需要一个空格，建议跟着这份文档手打食用更佳哦！）

  • 标题一共有6级，每增加一个#标题，字号越小，

  • 罗列点的时候，在文字前加上一个-或*或+，即有前方这个小圆点

    1. 标序号的罗列方法，数字和点加上空格，即如前方所示

  • 多次回车可以退出上述点和数字序号的换行自动编辑

调用注释或引用他人文字，在文字前加上>,即如前方所示

调用斜体，使用*或_，放在需要变斜体的左右

普通斜体 : 一个*或_

普通粗体：两个*或_

粗斜体 : 三个*或_
