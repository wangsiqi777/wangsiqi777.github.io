Python学习日记-01-字符串规整常用方法

2020年1月11日
| Python学习日记

1. Python格式要求

同一层次的语句必须有相同的缩进，每一组这样的语句称为一个块。

不要混合使用制表符和空格来缩进,因为这在跨越不同的平台的时候,无法正常工作。

！！！不要把word里的代码粘回编辑器，格式报错到跪下orz

2. 数据导入

常用存储方式：txt、CSV、Excel

CSV：第一行是列名，每一行用换行符号，每一列之间用逗号分割

txt：分隔符任意

Excel: 需要下载xlrd包，Python才会读取 excel

3. 数据规整

（1）字符串

  • split ( 分隔符，num )
分隔符：默认为所有空字符: 空格、换行（\n）、制表符（\t）; 也可以指定，如split(',')

num分割次数 : 默认-1，即分割所有

返回值 ：分割后的字符串列表

join是split的逆操作
  • replace( ‘旧字符串’ , ‘新字符串’ , ‘替换不超过多少次’ )
返回值：新的字符串，不会改变原字符串

##出现多个分隔符时，可以先用replace统一分隔符，再用split

str='i like apple,i like bananer'
print(str.replace(',', ' ').split(' '))

##结果

['i', 'like', 'apple', 'i', 'like', 'bananer']
  • strip( [chars] ) / lstrip( [chars] ) / rstrip ( [chars] )
[chars] : 表示字符串可选填，默认删除string头尾的空白符(包括 换行\n、回车\r、制表\t、空格' ')；
当chars不为空时，函数会被chars解成一个个的字符，然后将这些字符去掉。

该方法只能删除开头或是结尾的字符，不能删除中间部分的字符。

strip： 用来去除头尾字符、空白符(包括\n、\r、\t、' '，即：换行、回车、制表符、空格)
lstrip：用来去除开头字符、空白符(包括\n、\r、\t、' '，即：换行、回车、制表符、空格)
rstrip：用来去除结尾字符、空白符(包括\n、\r、\t、' '，即：换行、回车、制表符、空格)

返回值：新的字符串，不会改变原字符串

str = "123abcrunoob321"
print (str.strip( '12' ))  

##结果

3abcrunoob3

（2）字符串和列表互相转化

a = 'abcd efg'
b = list(a) #字符串列表化

c = ''.join(b) #列表字符串化

d = a.split() #split对单词列表化不是对每个字母

print('b is :',b)
print('d is :',d)
print('c is:',c)

##结果

b is : ['a', 'b', 'c', 'd', ' ', 'e', 'f', 'g']
d is : ['abcd', 'efg']
c is: abcd efg
[Finished in 0.5s]

（3）字符串的常见操作

  • 索引
正数是从0开始计数

负数是获取倒数第几个元素，从-1开始计数
  • 切片（一般不包括右边）
正着数：str[1:3] 获取偏移为1的元素...但不包括偏移为3

倒着数：str[-5:-3] 获取倒数第5个元素...但不包括倒数第3个元素

省略着数：str[1:] 获取偏移为1的元素...且包括到最后一个元素

        str[:] 获取所有元素

跳着数：str[::2] 步长为2，结果为srn
