深度学习结课项目——基于BP神经网络的Fashion-MNIST十分类问题（matlab）

2020年4月20日
|

1.数据集介绍

Fashion-MNIST数据集是由我们国家的一个大佬制作的，还发表了论文：Fashion-MNIST: a Novel Image Dataset for Benchmarking Machine Learning Algorithms paper 地址：https://arxiv.org/abs/1708.07747

数据集下载地址：[https://github.com/zalandoresearch/fashion-mnist][https://github.com/zalandoresearch/fashion-mnist]

Fashion-MNIST 数据集中的图片与衣着有关，其标签分别对应的衣着种类如表所示：

标签 代表的衣着种类
0 短袖圆领T恤
1 裤子
2 套衫
3 连衣裙
4 外套
5 凉鞋
6 衬衫
7 运动鞋
8 包
9 短靴 

其中值得我们关注的是数据制作的方法：
最原始图片是背景为浅灰色，分辨率为762x1000 的JPEG图片。然后经过resampled 到 51x73 的彩色图片。然后依次经过以下7个步骤，最终得到28x28的灰度图。
1. JPEG –> PNG
2. 裁剪背景
3. 按比例： 将图像缩放，也就是将一个维度缩放至28
4. 锐化
5. 再扩充至28*28，再把object调整至图片中央
6. 将负的像素点剔除
7. 转化成28x28的灰度图

本次实验的分类任务就是通过训练集训练后，对于给定一个样本，判断其所属的类别，对于训练集和测试集计算出整体的分类准确率，两者的准确率都应尽可能高。

2. 问题设计

BP算法是一种有监督的学习方法。BP 神经网络的重点是采用误差反向传播算法，不断更新网络权值。整个学习过程分为正向传播与误差的反向传播两个阶段。网络权值的调整是通过最小化网络输出与样本输出之间的误差平方和而训练出来的。

正向传播的过程中，网络的权值固定不变，每一层神经元的状态只影响下一层神经元的状态,这主要是由该层的权值与神经元相乘得到的结果形成下一层神经元的值，该过程在实现部分被定义为 fc 函

数。

反向传播过程中，未能满足要求的误差信号由输出端开始，逐层向前传播，并将误差分摊给各层的所有神经元，从而动态地调整各层神经元的连接权。这一过程在实现部分被定义为 bc 函数。

通过周而复始的正向传播与反向调节，神经元之间的权值不断被修正，当误差收敛时，停止学习。

具体的实现代码见第七部分。

3. 数据准备

在进行网络训练之前，需要对 Fashion-MNIST 数据集进行预处理。

  • train-images-idx3-ubyte.gz 训练集：存储了60000 个28x28 的灰度图样本，共有10 类

  • train-labels-idx1-ubyte.gz 中存储的是训练集的标签

  • t10k-images-idx3-ubyte.gz 测试集：存储了 10000 个测试样本

  • t10k-images-idx1-ubyte.gz 中存储的是测试集的标签。

Step1：对于灰度图样本，需要将 28*28 的样本转化为一个 784 维的列向量。

Step2：对于每一个样本，均需要进行归一化处理。采用min-max标准化方法，在实现部分对 784维的列向量中的每一个数值都除以255：(x - 0)/255。

Step3：对于标签，所给的标签以十进制 0-9 表示，采用one-hot编码，将其转化为一个 10×1 维的向量，每个样本中的特征位为 1,其余为 0。

4. 网络结构及训练参数设计

本实验采用5层BP网络设计，包括一个输入层，三个隐藏层，一个输出层，提取特征为28x28，每个特征样本数为60000，输入为28x28x60000；

经实验，当隐藏层神经元个数太大时，虽然BP网络越精确, 但会使收敛速度较慢，当较少时，则训练不够充分，准确率较低。每层神经元个数如下表所示：

表1：网络结构设计

层数 神经元个数
输入层（第一层） 784
第二层 256
第三层 128
第四层 64
输出层（第五层） 10

在网络训练的部分，使用梯度下降更新权值的方法。在训练时,首先将训练集打乱,每次取 1000 个样本及其标签进行训 练,如此循环使用整个测试集进行训练,先进行并将最终的权值保存。BP网络相关参数/函数设计如下表所示：

6. 分析与结论

BP神经网络的优点：

基于十分类图片任务的特性，本实验采用在图片分类任务上具有优良性能的BP神经网络。并在Fashion-MNIST数据集上对模型进行了训练和验证。总体来说，识别效果较好。

网络实质上实现了一个从输入到输出的映射功能，而数学理论已证明它具有实现任何复杂非线性映射的功能。这使得它特别适合于求解内部机制复杂的问题。

网络能通过学习带正确答案的实例集自动提取“合理的”求解规则，即具有自学习能力。

网络具有一定的推广、概括能力。

应用领域：

常见的有图像分类，自动驾驶，自然语言处理等。

BP神经网络的缺点：

对初始权重非常敏感，极易收敛于局部极小。

容易 Over Fitting 和 Over Training。

如何选择隐藏层数和神经元个数没有一个科学的指导流程，有时候感觉就是靠猜。

小结

本实验还存在很大的提升空间，因期末时间较紧没有继续尝试。如：从数学角度看，BP算法是一种梯度最速下降法，这就可能出现局部极小的问题。当出现局部极小时，从表面上看误差符合要求，但所得到的解并不一定是问题的真正解。针对此，国内外不少学者已提出了许多改进算法，如：可变学习速度的反向传播算法、学习速率的自适应调节等。

%%%%%%%%%%%%%%%%%main.m%%%%%%%%%%%%
%清理工作台
clear all;
clc;
 
%% Step 1: Data Preparation  
% loading dataset
% trainData: a matrix with size of 28x28x60000   
% trainLabels: a matrix with size of 10x60000
% testData: a matrix with size of 28x28x10000
% testLabels: a matrix with size of 10x10000
 
trainData = loadMNISTImages('train-images.idx3-ubyte');
labels = loadMNISTLabels('train-labels.idx1-ubyte');
    
% Transform the labels to correct target values.
trainLabels = 0.*ones(10, size(labels, 1));
    for n = 1: size(labels, 1)
        trainLabels(labels(n) + 1, n) = 1;
    end;
    
testData = loadMNISTImages('t10k-images.idx3-ubyte');
t_labels = loadMNISTLabels('t10k-labels.idx1-ubyte');
 
% Transform the labels to correct target values.
testLabels = 0.*ones(10, size(t_labels, 1));
    for n = 1: size(t_labels, 1)
        testLabels(t_labels(n) + 1, n) = 1;
    end;
 
train_size = 60000; % number of training samples
% input in the 1st layer 
X_train = reshape(trainData, 784, train_size); 
test_size = 10000; % number of testing samples
% external input in the 1st layer 
X_test = reshape(testData, 784, test_size); 
 
%% Step 2: Design Network Architecture
% define number of layers
L = 5; 
% define number of neurons in each layer 
layer_size = [784 % number of neurons in 1st layer
              256 % number of neurons in 2nd layer
              128 % number of neurons in 3rd layer
              64  % number of neurons in 4th layer
              10];% number of neurons in 5th layer
          
%% Step 3: Initial Parameters
% initialize weights in each layer with Gaussian distribution
for l = 1:L-1
    w{l} = 0.1 * randn(layer_size(l+1,1), sum(layer_size(l,:)));
end
 
alpha = 0.005; % initialize learning rate 
 
%% Step 4: Define Cost Function
% cost function is defined in cost.m
 
%% Step 5: Define Evaluation Index
% accuracy defined in accuracy.m
 
%% Step 6: Train the Network
J = []; % array to store cost of each mini batch
Acc = []; % array to store accuracy of each mini batch
max_epoch = 200; % number of training epoch
mini_batch = 100; % number of sample of each mini batch
 
figure % plot the cost
for iter=1:max_epoch
   % randomly permute the indexes of samples in training set
idxs = randperm(train_size); 
% for each mini-batch
for k = 1:ceil(train_size/mini_batch)
    % prepare internal inputs in 1st layer denoted by a{1}
    
    start_idx = (k-1)*mini_batch+1;          % start index of kth mini-batch
    end_idx = min(k*mini_batch, train_size); % end index of kth mini-batch
    a{1} = X_train(:,idxs(start_idx:end_idx));
    % prepare labels
    y = trainLabels(:, idxs(start_idx:end_idx));
    
        % forward computation
        for l=1:L-1
            [a{l+1}, z{l+1}] = fc(w{l}, a{l});
        end
 
        % Compute delta of last layer
        delta{L} = (a{L} - y).* a{L} .*(1-a{L}); %delta{L}={partial J}/{partial z^L} 
 
        % backward computation
        for l=L-1:-1:2
            delta{l} = bc(w{l}, z{l}, delta{l+1});
        end
 
        % update weight 
        for l=1:L-1
            % compute the gradient
            grad_w = delta{l+1} * a{l}'; 
            w{l} = w{l} - alpha*grad_w;
        end 
 
        % training cost on training batch
        J = [J 1/mini_batch*sum(cost(a{L}, y))];
        Acc =[Acc accuracy(a{L}, y)]; 
        % plot training error 
        plot(J);
        pause(0.000001);
    end
end 
% end training
% plot accuracy
figure
plot(Acc);
 
%% Step 7: Test the Network
%test on training set
a{1} = X_train;
for l = 1:L-1
  a{l+1} = fc(w{l}, a{l});
end
train_acc = accuracy(a{L}, trainLabels);
fprintf('Accuracy on training dataset is %f%%\n', train_acc*100);
 
%test on testing set
a{1} = X_test;
for l = 1:L-1
   a{l+1} = fc(w{l}, a{l});
end
test_acc = accuracy(a{L}, testLabels);
fprintf('Accuracy on testing dataset is %f%%\n', test_acc*100);
 
%% Step 8: Store the Network Parameters
save model.mat w layer_size

function labels = loadMNISTLabels(filename)
%loadMNISTLabels returns a [number of MNIST images]x1 matrix containing
%the labels for the MNIST images
fp = fopen(filename, 'rb');
assert(fp ~= -1, ['Could not open ', filename, '']);
magic = fread(fp, 1, 'int32', 0, 'ieee-be');
assert(magic == 2049, ['Bad magic number in ', filename, '']);
numLabels = fread(fp, 1, 'int32', 0, 'ieee-be');
labels = fread(fp, inf, 'unsigned char');
assert(size(labels,1) == numLabels, 'Mismatch in label count');
 
fclose(fp); 
End

%%%%%%%%%%%%%%%%%%%%%%loadMNISTImages.m%%%%%%%%%%%%%%%%%%%%
function images = loadMNISTImages(filename)
%loadMNISTImages returns a 28x28x[number of MNIST images] matrix containing
%the raw MNIST images
 
fp = fopen(filename, 'rb');
assert(fp ~= -1, ['Could not open ', filename, '']);
magic = fread(fp, 1, 'int32', 0, 'ieee-be');
assert(magic == 2051, ['Bad magic number in ', filename, '']);
 
numImages = fread(fp, 1, 'int32', 0, 'ieee-be');
numRows = fread(fp, 1, 'int32', 0, 'ieee-be');
numCols = fread(fp, 1, 'int32', 0, 'ieee-be');
 
images = fread(fp, inf, 'unsigned char');
images = reshape(images, numCols, numRows, numImages);
images = permute(images,[2 1 3]);
 
fclose(fp);
% Reshape to #pixels x #examples
images = reshape(images, size(images, 1) * size(images, 2), size(images, 3));
% Convert to double and rescale to [0,1]
images = double(images) / 255;
End

%%%%%%%%%%%%%%%%%%%%%%%%fc.m%%%%%%%%%%%%%%%%%%%%%%%%%
% This is forward computation function
function [a_next, z_next] = fc(w, a)
    % define the activation function
    f = @(s) 1 ./ (1 + exp(-s)); 
  
    % forward computing 
    z_next = w * a;
    a_next = f(z_next);
End

5)bc.m:
% This is backward computation function
function delta = bc(w, z, delta_next)
    % activation function
    f = @(s) 1 ./ (1 + exp(-s)); 
    
    % derivative of activation  function
    df = @(s) f(s) .* (1 - f(s)); 
 
    % backward computing
    delta = (w' * delta_next) .* df(z); 
End

%%%%%%%%%%%%%%%%%%%%%%%%%%cost.m:%%%%%%%%%%%%%%%%%%%%%%%
%% Step 4: Define Cost Function
function [J] = cost(a, y)
    J = 1/2 * sum((a - y).^2);
End

7)accuracy.m:
% Define Evaluation Index
function [ acc ] = accuracy( a, y )

    mini_batch = size(a, 2);
    [~,idx_a] = max(a);
    [~,idx_y] = max(y);
    acc = sum(idx_a==idx_y) / mini_batch;
End
