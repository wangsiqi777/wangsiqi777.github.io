数据挖掘-4-FP实战——AAAI-14 Accepted Papers标题分析

2020年4月30日
| 数据挖掘

1.实验目的：

读取[UCI] AAAI-14 Accepted Papers - Papers表格中title内容，利用Apriori关联算法分析标题中哪些单词的组合模式较为频繁，并找出最频繁的10项集。

2.数据集准备

①　观察[UCI] AAAI-14 Accepted Papers - Papers表格中的原始数据，一共有398条标题，同时还包括摘要、作者等信息。针对分析目标，首先使用wps自带功能删除无关行列，并将所有单词统一为小写形式，另存为txt。

②　按行读取存为list，根据空格分割字符串。

③　根据观察可知，标题中还包含-(),?:’特殊符号，需要消除。

④　在一条标题中，同一个单词重复出现只算做一次，需要消除一个list中的重复出现元素。

3.实验过程及结果分析

首先采用one-hot编码表示每一个单词在title中的出现情况。有多少个title就有多少位比特，每一个商品都有自己的编码状态，1表示出现单词在该比特位所代表的title中，0则代表没有。比如data只出现最后一个title中，一共五个title。那data就可以用[0,0,0,0,1]来表示

接着，使用apriori算法求解，它的核心思想是某个项集是频繁的，那么它的所有子集也是频繁的。因此，可以先遍历1个物品组合的情况，剔除掉支持度低于最小支持度的数据项，然后用剩下的物品进行组合。遍历2个物品组合的情况，再剔除不满足条件的组合。不断递归下去，直到不再有物品可以组合。求出不同的组合的频繁度。Python中提供了库函数apriori可以直接调用实现该算法。

该函数参数如下：

def apriori(df, min_support,use_colnames,max_len)

df：函数处理的数据集。
min_support：给定的最小支持度。
use_colnames：默认False，则返回的物品组合用编号显示，为True的话直接显示物品名称。
max_len：最大物品组合数，默认是None，不做限制。如果只需要计算两个物品组合的话，
便将这个值设置为2。

将相对支持度转换为绝对支持度，输出排名前10，如下图：

由图片可知，前10的全部为单个单词集且多为介词。虽然结果符合我们的认知，但结果没有太大的分析意义。因此，我扩大了范围，到前50的时候，可以看到更多具有分析意义的频繁词。

可以看到，在绝对频繁度前50中，具有实词意义的有：learning、multi、model、planning、（for,learning）、search、social、online、classification、analysis、information、sparse、models、（with，learning）、multiple、networks、（for，search）、supervised、robust、linear、probabilistic、games

大多词语与AI中的概念有关，如learning、classification、networks、supervised等等，符合AAAI大会的主题。

4.2 完整代码：

import pandas as pd
from mlxtend.preprocessing import TransactionEncoder
from mlxtend.frequent_patterns import apriori


#去除list中重复元素

def deleteDuplicatedElementFromList3(listA):
 return sorted(set(listA), key = listA.index)

# coding:utf-8

dataset = []

filename = "C://Users/LENOVO/Desktop/homework/Papers.txt"
with open(filename, "r") as f:
    Sum=0
    for line in f.readlines():
        data = line.replace(',','').replace(':','').replace('(','').replace(')','').replace('?','').replace('\"','').replace('-',' ').replace('\'',' ').split('\n')
        Sum=Sum+1
        for str in data:
            if str:
               sub_str = str.split(' ')
            dataset.append(deleteDuplicatedElementFromList3(sub_str))

te = TransactionEncoder()
#进行 one-hot 编码

te_ary = te.fit(dataset).transform(dataset)
df = pd.DataFrame(te_ary, columns=te.columns_)

#利用 Apriori 找出频繁项集

freq = apriori(df, min_support=0.005, use_colnames=True)
#换成绝对support

freq["support"] = freq["support"]*Sum
#输出排序后前十名

print(freq.sort_values("support",ascending = False).head(10))
