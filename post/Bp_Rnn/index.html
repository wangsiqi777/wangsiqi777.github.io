Bp_Rnn_LSTM

2019年4月29日
| 大创

BP

BP神经网络是线性权重的激活函数模型，即输入一个向量，对向量进行加权处理后输入到隐含层神经元的激活函数当中去，再将函数的输出值进行加权处理最后得到输出层的值。

使用梯度下降方法对参数进行优化。但是该方法在很多情形下无法达到全局最优解，只能达到部分最优解，容易出现“局部收敛”等问题，所以可以多次梯度下降去最值的方法进行最优化。

该种方法在识别前后没有关联的数据时方可奏效，但是在识别前后有关联的数据时，如果不把前一个数据作为后一个数据的输入的话，本质上该算法无法挖掘到数据产生的模式。基于挖掘时间序列数据内部模式的需求，我们创造了循环神经网络，使得输入之间是彼此有联系的。

RNN（Recurrent Neural Network）

RNN是BP的一种改进，前者BP神经网络的训练用一句话说就是：“我要寻找什么样的权重能使得输入加权在经过神经元函数作用再加权后的结果与实际结果最近。” 而RNN的训练用一句话说就是：“我要寻找什么样的权重使得输入和前一个输入加权经过函数作用再加权，再经过函数作用，再加权，再经过函数作用的结果与实际结果最近。”

  • 神经网络的信息流：

input -> hidden ->output

输入层循环

  • 记忆意味着隐藏层是，当前时刻的输入与隐藏层前一时刻的一种组合。

( input + prev_hidden ) -> hidden -> output

隐藏层循环

RNN中，神经元的输出可以在下一个时间戳直接作用到自身。

第i层神经元在m时刻的输入，除了（i-1）层神经元在该时刻的输出外，还包括其自身在（m-1）时刻的输出。

RNN可以看成一个在时间上传递的神经网络，它的深度是时间的长度！“梯度消失”现象又要出现了，只不过这次发生在时间轴上。

图中的context就是一个cell结构，可以看到它接受的输入有input(t)，context(t-1)，然后输出output(t)，比如像我们这个任务中，用到多层堆叠的rnn cell的话，也就是当前层的cell的output还要作为下一层cell的输入，因此可推出每个cell的输入和输出的shape是一样。如果输入的shape=(None, n)，加上context(t-1)同时作为输入部分，因此可以知道W的shape=(2n, n)。

别小看那一个小小的cell，它并不是只有1个neuron unit，而是n个hidden units。

LSTM

LSTM 通过不同的门来控制传输状态，这些门有“遗忘门”、“输入门”和“输出门”。通过这三个门的合作，LSTM 可以记住需要长时间记忆的重要信息，忘记不重要的信息；而不像普通的 RNN 那样只能够单调地仅有一种记忆叠加方式。因此对很多需要处理长序列数据的任务来说，LSTM-RNN 比 RNN有着更加优秀的表现。
