    • 学生教学评价 Page Rank 算法模型

1年1月1日

传统的教师教学质量评价体系中，教师教学质量包括学生评价 M1（30%）、系部评价 M2（30%）、督导评价 M3（20%）、教务处评价 M4（20%）。

每月底按照教师教学质量综合评分：Y=M1×30%+M2×30%+M3×20%+M4×20%计算出每位教师的综合得分。

在传统的教学评价体系中，M2 部分（系部评价）采取教学成绩为主要依据， M3 部分（教务处评价）以教学资料、教学进度为主要依据，这两部分相对客观占比约 50%。但 M1（学生评价信息）和 M3（督导评价）主观因素较多，误差很大。M1 部分要求学生认真、公正、准确地填写每门课程教师的应得分情况；M3 部分由督导深入课堂听课和教学秩序检查的形式进行考核，每月对所有任课教师进行经常性 听课、教学常规检查、教学资料检查、教学过程、教学环节、教学态度等进行全面考核。

学生对老师教学质量的评价，首先的前提是保证出勤，课堂是学习最主要的场所，然而很多学生对老师的评价都是在未到课堂或没有出勤足够课时而做出的评价，评价的结果显然是不客观的。督导的评价只是通过一两次听课的情况给出的，也存在不全面、不客观的现象。 本文基于 Google 的 Page Rank 网页排名算法思路[30]，设计了新的教学评价算法。在网页排名中，既要考虑网页被引用的数量，也要考虑引用网页的质量。同样，在教学评价中，也要兼顾对教学评价的学生人数和质量，排除无效的教学评价数据。

Page Rank 算法和学生教学评价的相关性与差异性

Page Rank 网页排名算法，根据网页之间相互的超链接作为网页排名的要素之 一。Google 把从 A 页面到 B 页面的链接解释为 A 页面给 B 页面投票，根据投票来 源和投票目标的等级来决定新的等级。一个高等级的页面可以使其他低等级页面 的等级提升。 学生教学评价类似于投票的过程，一名教师的教学水平客观体现在对学生的 吸引力上，假如所有学生在整学期的课时中出勤率都很高，那就足以证明教师的 课堂内容是有吸引力的；学生的每一次出勤都是对教师的一次投票。出勤还应区 别对待，虽然学生在课堂，但会存在人到心不到的现象，实际上这就是一次无效 的投票，还需要对投票做一个有效性评判。 学生的教学评价和 P age Rank 算法的相似之处是，两者都关注投票数量和每一 次投票的质量。但又有明显的差异：网页之间的投票都是相互的，但学生对教师 的投票确实仅仅由每个学生投向特定的一名教师，且学生之间不相互投票。因此 在算法上需要做出改变。

学生教学评价 Page Rank 算法模型

模型主要有如下几个参数：应到学生人数 S，课程总课时 T，整个课时学生的出勤状态 α。
